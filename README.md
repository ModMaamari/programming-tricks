### Check if a string is a palindrome
```is_palindrome = lambda x: x == x[::-1]```
Example:
```is_palindrome('racecar')```

### Swap two variables without a temporary variable
```a, b = b, a```
Example:
```a, b = 5, 10; a, b = b, a```
